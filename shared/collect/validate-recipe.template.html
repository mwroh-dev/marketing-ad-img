분석 레시피 확인 (validate-recipe)

각 광고를 우리가 어떻게 분석했는지 사실 그대로 보여줍니다. 시스템은 품질을 미리 판정하지 않습니다 — 분석이 맞는지 틀린지는 광고와 레시피를 직접 비교해 사람이 판단하세요. (에이전트가 확신하며 틀렸을 수도 있으니, 뱃지에 의존하지 말고 모든 카드를 보세요.)
바로잡으려면 카드의 📋 ID 버튼을 눌러 복사한 뒤, 터미널에서 에이전트에게 말하세요 — 예: "이 광고 재분석해줘".
※ 읽기 전용 로컬 화면입니다(내 컴퓨터에서만 실행, 일정 시간 뒤 자동 종료). 여기서 직접 고치지 않습니다 — 판단은 사람이, 수정은 터미널 대화로.
